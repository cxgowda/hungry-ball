A fast-paced browser game where you control a growing ball with your mouse or touch. Eat smaller balls to grow bigger, avoid bigger balls to survive, and try to achieve the highest score! Features smooth movement, dynamic ball generation, a glassy scoreboard, and responsive design for desktop and mobile.

Features:

Real-time smooth ball movement with mouse/touch controls

Balls of random size, color, and speed

Player grows by eating smaller balls, but touching larger balls ends the game

Increasing difficulty as player grows and speed increases

Glassy, transparent scoreboard displaying live score

High-score tracking (optional backend integration)

Fully responsive and mobile-friendly
